# Budget Calculator

Мини-репозиторий калькулятора личного бюджета на чистых HTML, CSS и JavaScript.

## Возможности

- Небольшой приветственный лендинг (hero, описание, CTA-кнопка, преимущества).
- Добавление операций с полями: тип, категория, сумма, дата, комментарий.
- Автоматический расчет доходов, расходов, баланса и процента сбережений.
- Удаление одной операции и полная очистка всех операций.
- Фильтрация операций по типу и категории.
- Экспорт всех операций в CSV.
- Сохранение данных в `localStorage`.
- Переключение темы: `light`, `dark`, `purple`.

## Структура

```text
budget-calculator/
├─ index.html
├─ css/style.css
├─ js/script.js
├─ README.md
└─ .gitignore
```

## Запуск

1. Откройте папку `budget-calculator`.
2. Запустите `index.html` в браузере.
3. Начните добавлять доходы и расходы.
